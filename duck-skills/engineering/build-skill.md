# Build Skill（造技能）

Design and build a new duck skill from scratch.

## Acceptance

- Starts from a user's idea or observed gap in existing skills
- Follows the skill spec format: title, one-liner, acceptance, implementation
- Writes the skill file to the correct directory under `duck-skills/`
- Skill is testable — can be invoked and produces verifiable output
- Matches the tone and conventions of existing skills
