# Guster-Ducks

You talk to Guster, and Guster manages the ducks. Ducks get things done.

## Layers

- `paths/` — Ducks exchange info through files.
- `hi-ducks/` — How you interact with ducks.
- `manage-ducks/` — How you manage ducks.
- `duck-skills/` — What ducks can do.

## Spec → Impl

`main` is specs only. Branch off to add your implementation.

## How to write

Follow the existing files as examples.
